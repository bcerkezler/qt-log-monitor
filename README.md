# QtLogMonitor

Qt Quick (QML) based desktop log viewer application.

## Current Version (v0.1)

- QML UI scaffold
- Search functionality
- Log level filtering (INFO / WARN / ERROR)

## Tech Stack

- Qt 6.x
- QML
- C++ (MSVC 2022)
- CMake

## Roadmap

- C++ backend (QAbstractListModel)
- Multithreaded log file reader
- Pause / Resume functionality
- File open dialog
